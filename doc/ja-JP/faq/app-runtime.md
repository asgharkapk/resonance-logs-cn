# FAQ · アプリの実行

## インストールと実行

### 設定はどう保存されますか？アップグレード後も残りますか？

新バージョンをインストールするたびに**ユーザー設定ディレクトリを削除しない**でください。設定は保持されます。今回の更新でフロントエンドに構造的な変更がなければ、設定は自動的に互換性を保ち保存されます。

### 0.0.7 初回利用時にデータベースのクリーンアップを促されますか？

以前 0.0.2 または 0.0.3 を使用していた場合、初めて 0.0.7 にアップグレードするときは旧データベースを手動で削除しないと異常が出ることがあります。

1. アプリを終了
2. `%LOCALAPPDATA%\resonance-logs-cn` 内の `resonance-logs-cn.db` を削除
3. アプリを再起動

## その他

### ログファイルはどこで取得できますか？

**DPSメーター → 設定 → デバッグ** で「ログを開く」からログディレクトリを開けます。

### 設定済みの構成を他人に共有するには？

ユーザー設定は `%APPDATA%\com.resonance-logs-cn` に保存されています。このディレクトリを圧縮して相手に渡し、相手は同じパスに展開すればあなたの設定を利用できます（操作前にアプリを終了してください）。

### オーバーレイウィンドウの切り替え方法は？

**DPSメーター → 設定 → ショートカット** で「オーバーレイウィンドウの切替」のショートカットを設定できます。

## データ取得

### WinDivert と Npcap、どちらを使いますか？

| 方式 | 説明 |
|------|------|
| **WinDivert** | 内蔵ドライバを使用。通常は Npcap の個別インストール不要。管理者権限が必要で、ウイルス対策ソフトに誤検知・隔離される場合があります |
| **Npcap** | [Npcap](https://npcap.com/) を別途インストールし、設定で正しい NIC を選択する必要があります |

設定パス：**DPSメーター → 設定 → ネットワーク**。

Npcap をインストールしたくない場合は WinDivert を利用できます。WinDivert が誤検知・隔離された場合は、Npcap に切り替えるか、アプリのインストール先と WinDivert 関連ファイルをセキュリティソフトの信頼リストに追加してください。

手順の詳細は [はじめに](../getting-started.md) を参照してください。

### データが出ない / キャプチャできない

1. ゲームが起動していることを確認
2. **DPSメーター → 設定 → ネットワーク** のキャプチャ方式を確認
3. Npcap 使用時：Npcap をインストールし、**ネットワーク** で正しいインターフェースを選択していることを確認
4. WinDivert 使用時：管理者権限で実行し、ファイアウォールやウイルス対策ソフトがブロック・隔離していないか確認
5. **アプリを完全終了してから再起動**（キャプチャ方式または NIC 変更後は再起動しないと反映されません）
6. 競合しうる VPN、プロキシ、その他のキャプチャツールを無効にして再試行

### WinDivert がファイアウォールやウイルス対策ソフトにブロックされた場合

WinDivert はネットワークキャプチャ用ドライバを読み込みます。一部のファイアウォールやウイルス対策ソフトはこれをリスク行為と判断する場合があります。ブロックまたは隔離されるとキャプチャが開始できず、リアルタイムデータが表示されないことがあります。

次の順で確認してください。

1. Windows ファイアウォールでアプリの通信を許可するか、一時的にファイアウォールを無効化してブロックが原因か確認
2. ウイルス対策ソフトの隔離領域で `WinDivert.dll`、`WinDivert64.sys`、アプリのインストール先が隔離されていないか確認
3. 火绒（Huorong）を使用している場合、**防护中心 → 病毒防护 → 信任区**（信頼リスト）でアプリのインストール先を信頼に追加。隔離領域に WinDivert 関連ファイルがある場合は、先に復元してから信頼に追加
4. アプリを完全終了してから再起動
5. 繰り返し隔離される場合は、Npcap キャプチャへの切り替えを推奨
