# Changelog v0.1.8

## 更新内容

### DPS 関連

- 特定ダメージを受けた味方を検知するための設定可能な警告ダメージを追加（DPS 検出 → 設定 → ライブ → チャレンジ監視（無効ダメージ））
  - 設定したダメージを受けた味方は、ライブ画面の名前の後ろに赤い三角の警告マークが表示されます
  - ワンクリック追加用のプリセット「N17-墓煌·浮游炮」を追加

### ライブモニター関連

- ローグのインスピレーション因子が因子エリアのカウントに影響する問題を修正
- 森語インスピレーション X2、X3、X5、X11、真因子 X2、X3 を追加
- 惩击スキル CD を追加

### モンスターモニター関連

- N20 ティナ攻撃低下イベント CD を追加
- 靭性バー表示を追加。モンスター buff やヘイトと同様、現在のメインターゲットを表示（モンスターモニター → 靭性バー）

### ダンジョンギミック関連

- 自身のキャラクター方向を表示する設定を追加。デフォルトはオフ（ダンジョンギミック → 自身マーカー → 方向インジケーター表示）
- 标点情報を表示する設定を追加。デフォルトはオフ（ダンジョンギミック → 标点表示）
- 巨塔ダンジョンギミックを追加（例：三花正しいテレポーター）
- ティナギミックを追加：ピザ危険区域警告、P2 点名順序
- 煌墓パズル点名表示を追加
- ナボ P2 ピザ危険区域警告を追加
- スキル表示ロジックを修正し、他スキルによってスキルグループ表示が中断されないように改善

## 注意事項

### 互換性に関する注意

- 0.0.2 または 0.0.3 を使用していた場合、0.0.4〜0.1.8 を初めて使う際は `%LOCALAPPDATA%\resonance-logs-cn` 配下の `resonance-logs-cn.db` を削除してからアプリを再起動してください
- 0.1.5 ではインスタンス保存 key（uid -> uuid）とパケットキャプチャ処理を変更しているため、異常があればログを添えて報告してください
- 0.1.5 で追加した因子の一部は十分に自測できていないため、異常があれば報告してください

### 特に重要

- コードを自分で改変し PR を出さずに配布する場合は、アプリ名・バージョン番号など原版の情報を変更してください
- 閉じるボタンは右下に非表示化されます。非表示バーの内容を右下へドラッグして表示できます
- よくある質問は使用マニュアルの HTML を参照してください
