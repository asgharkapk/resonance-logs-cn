# Changelog v0.1.7

## 更新内容

### DPS 関連

- 最新データを更新 @YozoraHoshi
- 龍機像の Boss 表示を削除

### ライブモニター関連

- CD 表示を改善。1/2 チャージ時も正常表示し、プログレスバーで充填時間を表示
- 炎角カウンターの異常を修正
- 幻想に常時表示スイッチを追加
- 治療ロボット CD を追加（ライブモニター → カスタム監視 → カウンター追加）
- 特定シーンでの自動表示/非表示設定を追加

### モンスターモニター関連

- Boss DBM を追加（Boss HP バー横のゲーム内ギミック字幕）
- 特定シーンでの自動表示/非表示設定を追加

### ダンジョンギミック関連

- ダンジョンギミック監視を新規追加。煌墓、S3 Raid、環海石礁（深海ナボ）に対応
  - 煌墓：半区点名、塔激活タイマー、突進安全区など
  - ナボ：イベント警告、波安全区など
  - S3 Raid：各種ギミック
- 特定シーンでの自動表示/非表示を追加。ダンジョンギミックは対応ダンジョンでのみ自動表示
- 詳細はダンジョンギミック設定を参照

### ドキュメント

- ダンジョンギミック、モンスターモニター Boss DBM の説明を追加

## 注意事項

### 互換性に関する注意

- 0.0.2 または 0.0.3 を使用していた場合、0.0.4〜0.1.7 を初めて使う際は `%LOCALAPPDATA%\resonance-logs-cn` 配下の `resonance-logs-cn.db` を削除してからアプリを再起動してください
- 0.1.5 ではインスタンス保存 key（uid -> uuid）とパケットキャプチャ処理を変更しているため、異常があればログを添えて報告してください
- 0.1.5 で追加した因子の一部は十分に自測できていないため、異常があれば報告してください

### 特に重要

- コードを自分で改変し PR を出さずに配布する場合は、アプリ名・バージョン番号など原版の情報を変更してください
- 閉じるボタンは右下に非表示化されます。非表示バーの内容を右下へドラッグして表示できます
- よくある質問は使用マニュアルの HTML を参照してください
