# モンスターモニター

ツールボックスの **モンスターモニター** に対応。ゲーム内の独立オーバーレイで**現在攻撃しているメインターゲット**の Buff、関連する味方 Buff、ヘイトリストを表示します。

**ライブモニター**（自身のスキル CD、自身 Buff など）とは独立。設定は「モンスターモニター」内に保存され、**モンスターオーバーレイ**ウィンドウで表示。ライブモニター のプリセットとは混在しません。

## 概要

- **現在攻撃メインターゲット**：オーバーレイの**モンスター Buff** と**ヘイトリスト**はキャラの**攻撃ターゲット**に自動追従（ゲーム内ターゲット ID）。`ExtraBuffMonitoredMonsters.json` の設定は不要（廃止済み）
- **切り替えタイミング**：別モンスターを選択して攻撃すると表示が切り替わる。ターゲット死亡または選択喪失後、対応 Buff / ヘイトはクリアされ、新しいメインターゲットまで待機
- **ダミー / 雑魚**：攻撃ターゲットになりゲームから同期されれば Boss と同様に表示。**ダミーを現在の攻撃ターゲットにする**（ダメージを与えるか選択）必要があり、近くにダミーがあるだけでは表示されません

> Buff 名が判別しにくい場合、各タブの **表示名称** で監視エントリにエイリアスを設定できます。

## 有効化とトップバー操作

- **モンスターモニターを有効化**：ページ上部マスタースイッチ。オフにするとモンスターオーバーレイへデータを送信しません
- **モンスターオーバーレイの切替**：ゲーム内 モンスターモニター オーバーレイの表示/非表示
- **モンスターレイアウトを編集**：オーバーレイ上で各領域（モンスター Buff エリア、味方 Buff エリア、ヘイトリストエリア、Boss DBM 区など）をドラッグ、リサイズ

モンスターモニターは複数の**設定プリセット**に対応しています。現在のロードアウトがその 1 つを参照し、ロードアウトの切り替え、またはモンスターモニターページでのプリセット切り替えにより、Buff、ヘイト、DBM、音声通知、レイアウトなどの設定がまとめて切り替わります。マスタースイッチ、デイリーシーンでの自動非表示、Buff 別名は引き続きグローバル設定で、プリセットには連動しません。

## ライブモニター との違い

| 項目 | ライブモニター | モンスターモニター |
|------|----------|----------|
| オーバーレイ | ゲームオーバーレイ（game-overlay） | モンスターオーバーレイ（monster-overlay） |
| 主対象 | 自身スキル、自身 Buff、属性など | 現在メインターゲット Buff、味方 Buff、ヘイト |
| プリセット | 複数の設定プリセットをロードアウトから参照・切り替え | 複数の設定プリセットをロードアウトから参照・切り替え |
| Buff 範囲 | プレイヤー自身関連 | モンスター身上 + 味方がモンスターへ付与 |
