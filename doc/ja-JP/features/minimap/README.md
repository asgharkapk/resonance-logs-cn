# ダンジョンギミック

ツールボックスの **ダンジョンギミック** に対応。対応ダンジョン内で、独立オーバーレイに站位・点名・ギミック警告をリアルタイム表示します。

**モンスターモニター**、**ライブモニター** とは独立。**ミニマップオーバーレイ** を使用し、対応シーンでのみ自動有効化されます。

## 概要

オーバーレイは 2 つの独立パネルで構成されます。

| パネル | 説明 |
|------|------|
| **マップパネル** | 俯瞰ミニマップ：自身、味方、Boss、ギミック関連ポイント。色・領域・線で頭割り、安全地帯、危険半区などを表示 |
| **ギミック字幕** | テキストリスト：進行中ギミック名、点名対象、残り時間 |

対応ダンジョンに入るとシーン ID から解析ロジックを読み込みます。未対応シーンではギミック内容は表示されません。

## 有効化とトップバー

**ダンジョンギミック** ページ上部：

- **ダンジョンギミックウィンドウの切替**：ゲーム内オーバーレイの表示/非表示
- **レイアウトを編集**：マップパネルとギミック字幕の位置・サイズを調整

## 表示オプション

| オプション | 説明 |
|------|------|
| **デイリーシーンで自動非表示** | デイリー黑名单シーンでは自動で非表示、退出後に以前の状態を復元 |
| **自身とギミック影響を受けた味方のみ表示** | 通常の味方ドットを非表示。点名や Buff 染色された味方は自動表示 |
| **Boss 位置を表示** | ミニマップ上に Boss を表示（既定オフ） |

## プレイヤー色と自身マーカー

- **プレイヤー色**：ギミック色がない場合の自身/味方の既定色
- **自身同心円**：点名で色が変わっても自身を素早く特定できる外周リング（色・線幅調整可）

## パネル表示

**マップパネル** と **ギミック字幕** を個別にオン/オフ。位置と幅はオーバーレイの **レイアウト編集** で調整。

## モンスターモニター Boss DBM との違い

| 項目 | ダンジョンギミック | モンスターモニター · Boss DBM |
|------|----------|---------------------|
| オーバーレイ | ミニマップオーバーレイ | モンスターオーバーレイ |
| 主情報 | 站位マップ + ギミック字幕 | Boss スキル警告バー（名称 + カウントダウン） |
| 適用範囲 | 対応ダンジョン | DBM イベントがある任意の戦闘 |

両方同時利用可能です。
