# Boss DBM（怪物监控）

对应 **怪物监控 → Boss DBM** Tab。

在怪物遮罩浮窗中显示 **Boss 技能机制预警条**：每条包含技能名称与剩余时间倒计时，类似 DBM 插件的计时条体验。

## 功能说明

- 数据来自游戏内 **Boss DBM 场景事件**（`SyncSceneEvents`），应用解析后推送到怪物遮罩。
- 技能名称按内置 **DBM 表**（`DbmTable.json`）匹配；未收录的条目会显示为 `#技能效果ID`。
- 多条机制同时进行时，按触发时间排序展示；每条带进度条表示剩余时间占比。
- **无需手动配置要监控的技能**：Boss 释放带 DBM 标记的机制时自动出现，机制结束后对应条目消失。

> Boss DBM 与 [Buff 监控](./buff.md) 互补：Buff 监控适合盯特定 Debuff / 层数；Boss DBM 适合 Boss 读条、阶段技能的统一倒计时条。

## 启用面板

1. 确认页顶 **启用怪物监控** 已打开。
2. 在 **Boss DBM** Tab 点击 **Boss DBM 区：显示**，或在 **启用窗口** Tab 中开启同一开关。
3. 顶栏 **切换怪物遮罩** 显示游戏内浮窗。

默认 **Boss DBM 区** 为关闭状态，避免与 Buff / 仇恨区域重叠；首次使用需手动开启。

## 样式调整

在 **Boss DBM** Tab 可调整 **Boss DBM 区** 专属样式（与怪物 Buff 区样式独立）：

| 项 | 说明 |
|----|------|
| 行间距 | 多条预警条之间的垂直间距 |
| 列间距 | 名称与倒计时之间的水平间距 |
| 字号 | 文字大小 |
| 名称颜色 / 数值颜色 | 技能名与倒计时文字颜色 |
| 进度条颜色 / 透明度 | 剩余时间进度条外观 |

## 布局编辑

顶栏 **编辑怪物布局** 进入编辑模式后：

- 拖动 **Boss DBM 区** 调整位置
- 拖拽右下角手柄缩放区域
- 无真实机制时显示 **机制预览** 占位条，便于提前摆位

编辑模式下可使用 **重置位置**、**重置尺寸**。

## 常见问题

**浮窗里没有 Boss DBM 条？**

1. 确认 **Boss DBM 区** 已设为显示（见上文）。
2. 确认 **启用怪物监控** 与 **切换怪物遮罩** 均已打开。
3. 当前战斗 Boss 须触发游戏内 DBM 事件；部分木桩或无 DBM 标记的技能不会出现条目。
4. 若应用在游戏进行中才启动，可 **切图** 一次触发状态同步（与实时监控相同）。

**与副本机制浮窗能否同时使用？**

可以。Boss DBM 在怪物遮罩，副本机制在独立浮窗，互不冲突。详见 [副本机制总览](../minimap/README.md)。
