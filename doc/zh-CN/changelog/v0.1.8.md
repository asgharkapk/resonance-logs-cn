# Changelog v0.1.8

## 更新内容

### DPS 相关

- 新增可配置的预警伤害，用于判断队友是否受到了某些伤害（DPS 检测 → 设置 → 实时 → 挑战监控（禁用伤害））
  - 队友受到设置的伤害列表会在 live 界面名称后面出现红色三角形感叹号用于提示
  - 预置 "N17-墓煌·浮游炮" 用于一键添加

### 实时监控相关

- 修复肉鸽的灵感因子会影响因子区计数的问题
- 新增森语灵感 X2、X3、X5、X11，真实因子 X2、X3
- 新增惩击技能 CD

### 怪物监控相关

- 新增 N20 蒂娜降攻事件 CD
- 新增韧性条展示，和怪物 buff 和仇恨一样，展示当前主目标的（怪物监控 → 韧性条）

### 副本监控相关

- 增加显示自身角色方向的配置，默认关闭（副本机制 → 自身标记 → 显示朝向指示）
- 增加显示标点信息的配置，默认关闭（副本机制 → 显示标点）
- 新增巨塔副本机制，例如三花正确传送门
- 新增蒂娜机制，披萨危险区预警，P2 点名顺序
- 新增煌墓拼图点名展示
- 新增娜宝 P2 披萨危险区预警
- 修复技能展示逻辑，避免会被其他技能打破技能组展示

## 注意事项

### 兼容性提醒

- 如果之前使用过 0.0.2 或 0.0.3，第一次使用 0.0.4-0.1.8 需要清理 `%LOCALAPPDATA%\resonance-logs-cn` 目录下的 `resonance-logs-cn.db`，再重新启动应用
- 0.1.5 由于修改了实例存储 key（uid -> uuid）、抓包逻辑，整体改动较大，若有异常请及时联系并提供日志
- 0.1.5 新增的因子有些没有进行自测（因子太贵了，并且没时间），如果有异常请及时反馈

### 特别注意

- 如果你自己修改了代码又不想提交 PR，在分享给其他人的时候请把应用名、版本号等原版相关信息都改掉，不要带着原仓库的信息，尤其是应用名和版本号
- 关闭按钮现在会隐藏到右下角，可以将隐藏栏的内容拖到右下角显示，避免窗口程序过多时无法快速找到
- 常见问题见使用手册中的 HTML
