# FAQ · 应用运行

## 安装与运行

### 如何保存设置？升级后设置会丢失吗？

每次安装新版本时**不要删除用户设置目录**，设置会保留。若本次更新前端无结构性调整，设置会自动兼容并保存；

### 0.0.7 首次使用提示清理数据库？

若你之前使用过 0.0.2 或 0.0.3，首次升级到 0.0.7 时需手动清理旧数据库，否则可能出现异常：

1. 关闭应用
2. 删除 `%LOCALAPPDATA%\resonance-logs-cn` 目录下的 `resonance-logs-cn.db`
3. 重新启动应用

## 其他

### 日志文件在哪获取？

在 **DPS检测 → 设置 → 调试** 中可「打开日志」目录

### 如何给别人分享自己已经配置好的配置？

用户配置存放在 `%APPDATA%\com.resonance-logs-cn` 目录下。将该目录打包发给对方，对方解压到相同路径即可使用你的配置（需关闭应用后操作）。

### 如何切换遮罩窗口？

在 **DPS检测 → 设置 → 快捷键** 中可设置「切换遮罩窗口」的快捷键。

## 数据捕获

### 抓包方式选 WinDivert 还是 Npcap？

| 方式 | 说明 |
|------|------|
| **WinDivert** | 内置驱动，通常不需要单独安装 Npcap；需要管理员权限，可能被杀毒软件误报或误隔离 |
| **Npcap** | 需单独安装 [Npcap](https://npcap.com/)，并在设置中选择正确网卡 |

设置路径：**DPS检测 → 设置 → 网络**。

若你不想安装 Npcap，可以使用 WinDivert；若 WinDivert 被杀毒软件误报、误隔离，建议改用 Npcap，或将应用目录和 WinDivert 相关文件加入安全软件信任区。

完整步骤见 [快速入门](../getting-started.md)。

### 没有数据 / 抓不到包怎么办？

1. 确认游戏已在运行
2. 检查 **DPS检测 → 设置 → 网络** 中的抓包方式
3. 若使用 Npcap：确认已安装 Npcap，并在 **网络** 中正确选择网络接口
4. 若使用 WinDivert：确认以管理员权限运行，并检查防火墙或杀毒软件是否拦截/隔离
5. **完全退出并重新启动应用**（更换抓包方式或网卡后必须重启才会生效）
6. 关闭可能冲突的 VPN、代理或其它抓包工具后重试

### WinDivert 被防火墙或杀毒软件拦截怎么办？

WinDivert 会加载网络抓包驱动，部分防火墙或杀毒软件可能将其识别为风险行为。若被拦截或隔离，应用可能无法启动抓包，表现为实时数据为空或启动后没有任何数据。

可按以下顺序排查：

1. 在 Windows 防火墙中允许应用通过，或临时关闭防火墙确认是否为拦截导致
2. 检查杀毒软件的隔离区，确认 `WinDivert.dll`、`WinDivert64.sys` 或应用安装目录是否被隔离
3. 若使用火绒，进入 **防护中心 → 病毒防护 → 信任区**，将应用安装目录加入信任；如果隔离区已有 WinDivert 相关文件，先恢复文件，再加入信任
4. 完全退出应用后重新启动
5. 若仍然反复被误隔离，建议切换为 Npcap 抓包
