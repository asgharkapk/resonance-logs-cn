# Changelog v0.1.7

## Changes

### DPS

- Latest data updates @YozoraHoshi
- Removed Dragon Machine Boss display

### Live Monitor

- Improved CD display for 1/2-charge skills; progress bar shows recharge time
- Fixed Flame Horn counter anomaly
- Fantasy: added always-on display toggle
- Added healing bot CD counter (Live Monitor → Custom Monitor → Add Counter)
- Added auto show/hide in specific scenes

### Monster Monitor

- Added Boss DBM (in-game mechanic bars next to the Boss health bar)
- Added auto show/hide in specific scenes

### Dungeon Mechanics

- New dungeon mechanics overlay for Cursed Tomb, S3 Raid, and Sea-Ringed Reef
  - Cursed Tomb: half-area callouts, tower activation timer, charge safe zones, etc.
  - Sea-Ringed Reef: event warnings, wave safe zones, etc.
  - S3 Raid: various mechanics
- Auto show/hide in specific scenes; dungeon mechanics only appear in supported dungeons
- See Dungeon Mechanics settings for details

### Documentation

- Added guides for dungeon mechanics and Monster Monitor Boss DBM

## Notes

### Compatibility

- If you used 0.0.2 or 0.0.3, first launch on 0.0.4-0.1.7 requires deleting `resonance-logs-cn.db` under `%LOCALAPPDATA%\resonance-logs-cn` and restarting
- 0.1.5 changed instance storage keys from uid to uuid and refactored packet capture; report logs if issues occur
- Some factors added in 0.1.5 were not fully self-tested; please report abnormal behavior

### Important

- If you fork without submitting a PR, change app name, version, and upstream branding before sharing
- Close button hides to bottom-right; drag hidden bar contents to bottom-right to show them
- FAQ is available in the manual HTML
