# Changelog v0.1.8

## Changes

### DPS

- Added configurable alert damage to detect when teammates take certain damage (DPS Settings → Live → Challenge Monitor → Disabled Damage)
  - Teammates taking configured damage show a red triangle exclamation mark after their name on the live screen
  - Preset "N17-Cursed Tomb Floating Cannon" for one-click add

### Live Monitor

- Fixed rogue inspiration factors affecting factor area count
- Added Verdant Oracle inspiration X2, X3, X5, X11; true factors X2, X3
- Added Smite skill CD

### Monster Monitor

- Added N20 Tina Attack Down event CD
- Added toughness bar display; like monster buffs and aggro, shows the current main target (Monster Monitor → Toughness Bar)

### Dungeon Mechanics

- Added option to show own character direction, off by default (Dungeon Mechanics → Self Marker → Show Direction Indicator)
- Added option to show waypoint info, off by default (Dungeon Mechanics → Show Waypoints)
- Added Tower dungeon mechanics, e.g. correct three-flower teleporter
- Added Tina mechanics: pizza danger zone warning, P2 callout order
- Added Cursed Tomb puzzle callout display
- Added Sea-Ringed Reef P2 pizza danger zone warning
- Fixed skill display logic to prevent other skills from breaking skill group display

## Notes

### Compatibility

- If you used 0.0.2 or 0.0.3, first launch on 0.0.4-0.1.8 requires deleting `resonance-logs-cn.db` under `%LOCALAPPDATA%\resonance-logs-cn` and restarting
- 0.1.5 changed instance storage keys from uid to uuid and refactored packet capture; report logs if issues occur
- Some factors added in 0.1.5 were not fully self-tested; please report abnormal behavior

### Important

- If you fork without submitting a PR, change app name, version, and upstream branding before sharing
- Close button hides to bottom-right; drag hidden bar contents to bottom-right to show them
- FAQ is available in the manual HTML
